# 🌦️ SkyCast - Modern React Weather Application

SkyCast is a sleek, responsive, and feature-rich weather dashboard built with **React 19** and **Vite 7**. It provides real-time weather updates, hourly forecasts, and 7-day predictions with a premium glassmorphism design.

![Weather App Preview](https://raw.githubusercontent.com/username/repo/main/preview.png) *(Add your own screenshot here)*

## ✨ Features

- **🔍 Smart Search:** Instant weather lookup for any city worldwide.
- **🌡️ Real-time Data:** Current temperature, humidity, and wind speed.
- **📅 Extended Forecast:** Comprehensive 7-day weather outlook.
- **⏰ Hourly Updates:** Plan your day with detailed hourly transitions.
- **🎨 Premium UI:** Modern dark theme with glassmorphism effects and smooth gradients.
- **📱 Fully Responsive:** Optimized for desktop, tablet, and mobile screens.

## 🚀 Tech Stack

- **Framework:** [React 19](https://react.dev/)
- **Build Tool:** [Vite 7](https://vitejs.dev/)
- **Styling:** Custom CSS (Glassmorphism, Flexbox/Grid)
- **APIs:** 
  - [OpenWeatherMap](https://openweathermap.org/api) (Current Weather)
  - [Open-Meteo](https://open-meteo.com/) (Forecast Data)
- **Typography:** [Inter](https://fonts.google.com/specimen/Inter) via Google Fonts

## 🛠️ Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/your-username/skycast.git
cd skycast/frontend
```

### 2. Install dependencies
```bash
npm install
```

### 3. Environment Variables
Create a `.env` file in the `frontend` root directory and add your OpenWeatherMap API key:

```env
VITE_API_KEY=your_openweathermap_api_key_here
```

### 4. Run the development server
```bash
npm run dev
```

The app will be available at `http://localhost:5173`.

## 📦 Scripts

- `npm run dev` - Starts the development server.
- `npm run build` - Builds the app for production.
- `npm run lint` - Runs ESLint to check for code quality issues.
- `npm run preview` - Previews the production build locally.
